# Ideas Vault — 100 Top-Tier Web3 Ideas for Arbitrum Open House 2026

**100 Orbital-tier ideas** across 8 categories, each with a full file: pitch, what it is, how it works, why it lands, future utility, business model, sponsor fit, 3-week MVP scope, risks, and verified sources.

Built by an 8-teammate research team (`ideas-100`) over a single pass — each teammate owned one disjoint category folder, applied the Orbital filter independently, and pushed cuts into per-category `discarded.md` files.

---

## The filter that built this list

Every pitch ≤15 words must trigger *"wait, how is that possible?"* in a smart non-crypto listener — same shock as Orbital Pool's *"Uniswap is 1D, Curve is 2D, Orbital is N-D liquidity across N stablecoins."* No padding. Each writer was instructed to ship fewer ideas rather than weaken the bar.

## How to use this vault

1. **First pass (~30 min):** read the pitch tables above. Mark the 5-10 that hit you in the gut. Don't think about feasibility.
